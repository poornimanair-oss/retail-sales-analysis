# Retail Sales Analysis

## Tools Used
- Python
- Pandas
- Excel
- Power BI

## Project Objective
Analyze retail sales data to identify customer purchasing patterns and revenue trends.

## Key Insights
- Electronics generated the highest revenue.
- Clothing had the highest purchase frequency.
- Female customers contributed more purchases.
- Customers aged 45–55 were the highest spenders.
- Sales peaked in May and were lowest in September.
